# Extract Log-Likelihood of galamm Object

Extract Log-Likelihood of galamm Object

## Usage

``` r
# S3 method for class 'galamm'
logLik(object, ...)
```

## Arguments

  Object

- ...:

  Other arguments

## Examples

``` r
# Linear mixed model with heteroscedastic residuals
mod <- galamm(
  formula = y ~ x + (1 | id),
  dispformula = ~ (1 | item),
  data = hsced
)

# Extract log likelihood
logLik(mod)
#> 'log Lik.' -2058.14 (df=5)
```
