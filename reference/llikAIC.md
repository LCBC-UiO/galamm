# Extract log likelihood, AIC, and related statistics from a GALAMM

This function is assembles the values used by
[`summary.galamm`](https://lcbc-uio.github.io/galamm/reference/summary.galamm.md).

## Usage

``` r
llikAIC(object)
```

## Arguments

- object:

  Object of class `galamm` returned from
  [`galamm`](https://lcbc-uio.github.io/galamm/reference/galamm.md).

## Value

A list containing AIC, BIC, log likelihood, deviance and residual
degrees of freedom.
