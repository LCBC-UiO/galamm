# Example Data with Heteroscedastic Residuals

Simulated dataset with residual standard deviation that varies between
items.

## Usage

``` r
hsced
```

## Format

### `hsced` A data frame with 1200 rows and 5 columns:

- id:

  Subject ID.

- age:

  Timepoint.

- item:

  Item indicator.

- x:

  Explanatory variable

- y:

  Outcome.
